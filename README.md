### 🚀 Projects & Open Source

* 🐾 **[Pawdig](http://www.pawdig.com):** Building an AI document intelligence tool that extracts messy data from documents like PDFs, invoices, and contracts, turning files into a private, searchable AI knowledge base.
* 🧠 **[ragviz](https://github.com/zhijiewong/ragviz):** A Graph RAG visualization platform featuring interactive knowledge graphs and real-time parameter tuning for local LLMs. 
* ⚡ **[viber](https://github.com/zhijiewong/viber):** A lightweight DOM capture utility built specifically to empower vibe coders.
* 🤖 **Open-Source AIOps Platform (Coming Soon):** Building an open-source platform that applies artificial intelligence, machine learning (ML), and natural language processing (NLP) to automate and streamline IT operations.
